This is a simple web program to randomly generate a week's exercise plan.

First choose the types of exercise you might be up for in the coming week.

Walking | Jogging | Swimming | Cycling | Cardio in the gym | A random cardio exercise class | A pilates or yoga class | Weight training | A youtube pilates vid | A youtube cardio vid
Select/Deselect all exercise types

Then select how much time each day you could do exercise

The "randomiser" assumes that classes have to last 60 minutes so if you're hoping to get some classes come up be sure to include some 60 minute slots.

Write in an activity if you want to over-ride the randomiser for a particular day; leave it as "Random" if you want the randomiser to choose.

Monday: of
Tuesday: of
Wednesday: of
Thursday: of
Friday: of
Saturday: of
Sunday: of

And finally, click the button to generate a random plan. If you don't like it, just click the button again.
